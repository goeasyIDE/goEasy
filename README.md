# openIDE

`openIDE` is a lightweight, web-based IDE that allows you to run, manage, and debug code in multiple languages directly from your browser. It supports extension installation, terminal commands, live process management, and interactive CLI prompts, making development seamless across platforms.

![openIDE Logo](img/icon.png){width="64px"}

---

## Features

- Run code in multiple languages:  
  <img src="img/js.png" width="24px" /> <img src="img/py.png" width="24px" /> <img src="img/nodejs.png" width="24px" /> <img src="img/ts.png" width="24px" /> <img src="img/rb.png" width="24px" /> <img src="img/php.png" width="24px" /> <img src="img/cpp.png" width="24px" /> <img src="img/java.png" width="24px" /> <img src="img/kt.png" width="24px" /> <img src="img/go.png" width="24px" /> <img src="img/icon.png" width="24px" />

- Compile and run compiled languages with real-time output.
- Manage running processes and kill them directly from the web interface.
- Install and uninstall terminal extensions with proper command queues.
- Supports interactive CLI prompts (`y/n` and text input).
- File and folder management directly from the IDE.
- Multi-platform support: Windows, Linux, macOS.
- Fully extensible via terminal extensions.
- Type `openIDE` in any folder to set it as source folder for openIDE, and it will start the IDE and start accessing your terminal through that folder
