# openIDE

`openIDE` is a lightweight, web-based IDE that allows you to run, manage, and debug code in multiple languages directly from your browser. It supports extension installation, terminal commands, and live process management, making development seamless across platforms.

![openIDE Logo](img/icon.png)

---

## Features

- Run code in multiple languages:  
  ![JavaScript](img/js.png) ![Python](img/py.png) ![Node.js](img/nodejs.png) ![TypeScript](img/ts.png)
- Compile and run compiled languages with real-time output.
- Manage running processes and kill them from the web interface.
- Install and uninstall terminal extensions with command queues.
- Support for interactive CLI prompts (`y/n`, input prompts).
- File and folder management directly from the IDE.
- Multi-platform support: Windows, Linux, macOS.
- Fully extensible via terminal extensions.
