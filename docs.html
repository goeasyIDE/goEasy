What is openIDE?

openIDE is a lightweight web-based IDE designed mainly for mobile users who want to code, run, and manage projects directly from their browser. It provides terminal access, extension support, and multi-language execution — all without needing a heavy desktop setup.

How to use openIDE?

To start using openIDE, you need Node.js installed.

For mobile users:

  1. Install a TTY app like Termux
  2. Run:
pkg install nodejs

Then install openIDE globally:

npm install -g @goeasyide/openIDE

Now go to your source/project folder and run: 

openIDE

This will start the openIDE server from that folder.

You will be asked to enter a password for the first time. Make sure to use a very strong password. We are not responsible for any casualties caused by weak security.

How to make your own extension?

  1. Join our Discord server: https://discord.gg/v3gcb7ptXg
  2. Apply there to get access for extension development.
  3. For any issues, contact staff on Discord.
  4. Check our GitHub to understand the blueprint of how extensions are built.

How can your extension be banned or cancelled?

  • Your extension can be cancelled due to inappropriate name or description.
  • Only valid types are iframe and terminal. Others will be rejected.
  • Using inappropriate URLs, websites, or non-iframe-compatible pages will lead to cancellation.
  • Inappropriate icons are not allowed.
  • Using openIDE routes in backend is strictly prohibited and can lead to PERMANENT BAN.
  • Sending or storing user credentials (IP, password) is strictly prohibited and leads to PERMANENT BAN.
  • Changing extension version or code without proper update application leads to PERMANENT BAN.
  • Terminal extensions installing/uninstalling harmful or bad packages will be cancelled.
